Pasos seguidos para la creación de esta web

Para realizar esta página web en primer lugar se descargó pandoc utilizando: brew install pandoc. Una vez realizado este paso se descargaron de internet los ejemplos de Bootstrap para utilizarlos como plantillas de estilo de HTML. Después de esto se ha creado una carpeta llamada "docs" dentro del repositorio conectado con GitHub. Tras descargar la carpeta Bootstrap se ha hecho "unzip" para poder extraer los archivos necesarios.

Después de esto se ha utilizado el modelo sicky-footer-navbar. A través del comando mv se ha trasladado el archivo css a la carpeta de docs (dentro de una carpeta css) y posteriormente de bootstrap se ha tomado el archivo bootsrap.bundle.min.js y también se ha trasladado a la carpeta docs (dentro de una carpeta js "javascript".

El modelo sticky-footer-navbar contenía un archivo llamado "index.html" que finalmente era la plantilla sobre la que escribir el texto deseado y adjuntar imágenes. Además, se ha creado también a través de nano un archivo "cabecera.html" y "pie.html. No obstante, como son varias prácticas se han creado diferentes archivos html. Cada archivo se ha titulado según el número de la práctica. Las prácticas realizadas durante el curso se han convertido (con la herramienta pandoc) de formato md a html y han sido pegadas en esta plantilla justo debajo de Begin Content". Una vez que se ha pegado el contenido se han editado ciertos aspectos estilísticos se han guardado estos archivos.
