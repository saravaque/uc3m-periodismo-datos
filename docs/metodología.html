Pasos seguidos para la creación de esta web

Para realizar esta página web en primer lugar se descargó la herramienta Pandoc. Esto serviría posteriormente para traducir los archivos que se encontraban en mi repositorio en formato MarkDown a HTML. Para ello utilicé “brew Install pandoc”. El siguiente paso fue conseguir la plantilla para poder crear la página web. Ello se hizo gracias a una biblioteca de ejemplos llamada Bootstrap. Descargué todos los ejemplos desde la terminal de la misma forma que Pandoc y posteriormente tuve que descomprimir la carpeta con unzip.

Tras esto con el comando mv, trasladé índex.html y sticky-footer.header.css a docs. El archivo .css a su vez fue trasladado a una carpeta creada (con “mkdir”) llamada CSS a través de “mv2”. Así mismo también trasladé el archivo bootstrap.bundle.min.js (javascript) a otra carpeta creada dentro de docs llamada js.

Posteriormente he creado varios archivos .html: + cabecera.html + pie.html + Práctica1.html + Práctica2.html + Práctica3.html + Metodología.html

En cada uno de estos archivos bajo “Begin page content” se ha pegado el texto que correspondía. Previamente traducido a lenguaje html con la herramienta Pandoc.

El “header” se ha editado también en cada archivo html, para que apareciesen los accesos a las demás prácticas (páginas) y para que se pudiera retornar al inicio (situado en el índex.html).
